# Rock Paper Scissors Game

# Description
This is a fun Java game where you can play Rock, Paper, Scissors â€” with two extra choices: Lizard and Spock!

You play against the computer. After each round, the game shows who won and keeps the score. You can play as many times as you like, or type `exit` to quit.

# Rules
Here are some rules for the game.
- Scissors cuts Paper
- Paper covers Rock
- Rock crushes Lizard
- Lizard beats Spock
- Spock breaks Scissors
- Scissors beats Lizard
- Lizard eats Paper
- Paper beats Spock
- Spock breaks Rock
- Rock beats Scissors
